# Diamont

A Flutter project to display a diamond.

## Dependencies

- Lottie - To display the diamond's animation
- Splashscreen - Change the loading screen before app is ready (pre setted to 3 seconds, could be lower)

## Getting Started

- Clone the repository
- run: pub get
- run: flutter run
